မြန်မာနိုင်ငံရှိ မြို့အားလုံး အတွက် API

မြို့ပေါင်း (၅၀၀) ကျော် ဒေတာများဖြင့် မြန်မာဘာသာဖြင့် Data များရရှိပါမည်။

အသုံးပြုရန်အတွက် API Key မလိုအပ်ပဲ လွယ်ကူစွာ အသုံးပြုနိုင်ပါသည်။

ဒေတာများကို Google Map မှ ယူထားခြင်းဖြစ်သည်။

ယခု Script သည် G Suite Free Edition ဖြင့် Run ထားသောကြောင့် Limitation ရှိနိုင်ပြီး Response Time အနည်းငယ် ကြာနိုင်ပါသည်။
လုပ်ငန်း ဆိုင်ရာ App နှင့် Web များ တွင် အသုံးပြုရန် အတွက် ကိုယ်ပိုင် Script နှင့် Dataset လိုအပ်ပါက ကျွန်တော်ထံသို့ ဆက်သွယ်နိုင်ပါသည်။
